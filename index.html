Guía Rápida de Instalación (PC / Laptops)

  • Presiona el botón superior para descargar el paquete comprimido CATALOGO.VIRTUAL.zip.
  • Haz clic derecho sobre el archivo descargado y selecciona "Extraer todo...".
  • Abre la nueva carpeta extraída y ejecuta la aplicación .exe que se encuentra dentro.
⚠️ IMPORTANTE: Este sistema es exclusivo para Windows. No separes el archivo .exe de sus carpetas de imágenes, de lo contrario la interfaz virtual no cargará los fósiles.
